# Webpack Sass Boilerplate
## Installation
1. Clone this repository
2. `npm install`
## Usage
### Development server
```
npm start
```
Development server located at `localhost:8000`.
### Build
```
npm run build
```
Build output directory: `/dist`
## Features
* [Webpack 5](https://webpack.js.org/)
* [Sass](https://sass-lang.com/)
